1. Installo la libreria tramite il comando (utilizziamo npm) 'npm install primeng @primeuix/themes'
2. Installo le icone di PrimeNg con il comando 'npm install primeicons'
3. Posso eseguire i due passaggi precedenti in un unico passaggio tramite il comando 'npm install primeng @primeuix/themes primeicons'
